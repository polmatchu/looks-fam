☁ Cloud Sync via GitHub Gist

Your question bank is saved to a private GitHub Gist. Any device that opens this app and has the same token configured will automatically load the latest questions. Free, private, no extra accounts needed.
1

Go to github.com/settings/tokens/new, create a Classic token with the gist scope only. Copy the token.

2

Paste it below and click Connect. A private Gist will be created automatically to store your questions.

3

On any other device, open this same app URL, enter the same token and the Gist ID shown after connecting, then click Load from Cloud.
